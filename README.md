# SQL Practice

This repo contains SQL practice questions that were originally done by hand and then written in SQL.

## Tables

* Students
* Courses
* Enrollments
* Salaries
* Projects

## How I Did It

1. Read each question carefully
2. Solved it on paper first
3. Identified what was needed (filter, group, aggregate, etc.)
4. Wrote the SQL query
5. Checked results with the data

## What I Practiced

* SELECT and WHERE
* IN, BETWEEN, NOT IN
* GROUP BY
* HAVING
* Aggregate functions (SUM, AVG, COUNT, MIN, MAX)
* ORDER BY and LIMIT

## Goal

To improve SQL skills by understanding the logic before writing queries.

## Notes

* WHERE filters rows
* GROUP BY groups data
* HAVING filters grouped data
